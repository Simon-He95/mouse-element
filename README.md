## mouse-element
提供一个简单的鼠标小动画

## 🌈 Usage
```vue
<main w-full h-full @mousemove="e => createElement(e, { backgroundColor: 'yellow', timeout: 1000 })">
</main>
```

## 🚀 Props
- backgroundColor: 背景颜色
- timeout: 动画时长
- width: 宽度
- height: 高度
